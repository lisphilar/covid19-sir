Term definitionÂ¶

This is a list of term in CovsirPhy project. These terms are used as method names and figure titles.

model

ODE models derived from the simple SIR model.

accuracy of models

Whether variables and parameters of a model are represent the currenct situation of the outbreak or not.

phase

A sequential dates in which parameter values of SIR-derived models are fixed.

trend

Parameter set of a phase.

trend analysis

Breaking down the series of records to phases.

accuracy of trend analysis

Whether the change date of phases are effective or not.

parameter estimation

Optimization of model parameter values with actual records of a phase.

accuracy of parameter estimation

How small the difference of actual number of cases and simulated number of cases with the estimated parameter values in a phase. This will be scored with RMSLE.

simulation

Calculation of the number of cases with fixed parameter values and initial values.

prediction

Prediction of parameter values in the future phases using relationship of estimated parameters and indexes regarding measures taken by countries and individuals.

forecasting

Forecasting of the number of cases in the future phases. This needs trend analysis, parameter estimation, simulation and prediction.
