WorkflowÂ¶

The easiest way to know the usage of covsirphy core functionalities is to read Usage: quickest tour after instalation of the latest version.

Then, please take a look at the following. Here is the workflow of analysis. They can be done with a few lines of Python codee and find the details with the linked documents.

  1. Prepare datasets

    1. Decide whether to use the recommended datasets

    2. (Optional) Read local datasets

    3. (Optional) Perform database lock

    4. Download the recommended datasets

    5. Clean data

  2. Perform exploratory data analysis

  3. Learn SIR-derived models

  4. Learn S-R trend analysis

  5. Perform scenario analysis

    1. Register cleaned data

    2. Check records of the selected country/province

    3. Perform S-R trend analysis to split time series data to phases

    4. Estimate ODE parameter values in the past phases

    5. (Experimental) Predict ODE parameter values in the future phases

    6. Simulate the number of cases with some scenarios

    7. (To Be Implemented) Find solutions to end the outbreak

If you have any questions and any advice, feel free to contact developers and users via GitHub Issues. Always welcome!
